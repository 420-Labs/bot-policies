Privacy Policy

We will never store any data unless the user agrees (also known as "creating a profile" and completing the "Registration") to do so by clicking "I agree" button during the creation of a profile in Bubble. We explicitly do not store any data that can lead to the identification of an individual outside Discord

What data do we collect

In order to provide the complete experience of the Service, we need to collect and user information from our users.

The information we collect includes:

  • Discord User ID (To link the messages and the settings to the user)
  • The settings the user has set for the bot, (e.g. Fluent Chat Timeout)
  • Messages the user had with the bot (read more here)
  • Logs in case of an Error, this helps us fixing the error

Why we store your messages?

We need to store messages you have with the bot in order to provide you the experience of having a chat that understand past messages, for example, you are talking with the bot about a specific Computer, without context, you would have to repeat that you are talking about that computer. Simulation:

Storing messages enabled:

  • User: Hey, what do you think about the computer (the computer name)
  • Bubble: Hey! Yes, i know it, it is a very interesting computer. What do you want to know about it?
  • User: Do you know the RAM memory it has?
  • Bubble: Yes, it has 16GB RAM DDR4 at 3200Mhz
